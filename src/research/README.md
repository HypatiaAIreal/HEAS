# Research Components

This directory handles the actual research execution.

## Components

- **SearchEngine.ts** - Wrapper around web_search tool with smart query generation (Phase 1)
- **PaperAnalyzer.ts** - Downloads and analyzes academic papers (Phase 5)
- **KnowledgeExtractor.ts** - Extracts insights using NLP (Phase 2)
- **SynthesisEngine.ts** - Synthesizes findings into structured summaries (Phase 2)

## Flow

```
SearchEngine → PaperAnalyzer → KnowledgeExtractor → SynthesisEngine
```
